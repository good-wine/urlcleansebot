# Support

## Getting Help

- **Bug reports & feature requests** — Open a [GitHub Issue](https://github.com/good-wine/urlcleansebot/issues/new/choose)
- **Security disclosures** — See [SECURITY.md](SECURITY.md)
- **Documentation** — Check [README.md](README.md) and [docs/](docs/) first

## Community

- **GitHub Discussions** — Ask questions and share ideas
- **Telegram** — Contact the bot maintainer directly

## Before Opening an Issue

1. Check the [FAQ](docs/DEPLOYMENT.md#troubleshooting) and existing issues
2. Run with `RUST_LOG=debug` and include relevant logs
3. Specify your environment: Rust version, OS, database type, deployment method
